# HR Analytics Dashboard 📊

## Overview
This Power BI dashboard analyzes employee attrition and workforce trends.

## Key Insights
- Attrition Rate: 16.1%
- Highest attrition in 26–35 age group
- Salary slab "Up to 5k" has highest attrition
